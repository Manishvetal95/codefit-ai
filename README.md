# CodeFit AI: The Engineering Excellence Engine 🚀

> **Master the Logic. Own the Discipline.**  
> *A high-performance AI Mentor for Computer Engineering students bridging the gap between Data Structures and Physical Fitness.*

---

## 💡 The Why
Most engineering students burn out between complex Algorithmic logic and sedentary lifestyles. **CodeFit AI** is built to solve this dual challenge. It’s not just a chatbot; it’s a discipline-first ecosystem designed for the next generation of engineers.

## 🏛️ Core Pillars

### 1. The Innovation Dashboard 📊
A GitHub-style activity heatmap that tracks your dual consistency. 
- **Coding Streaks**: Track every logic session.
- **Fitness Streaks**: Monitor your physical discipline.
- **XP System**: Level up as you master new concepts.

### 2. The Sprint Mechanic ⚡
A productivity-first widget integrated directly into your workflow.
- **25m Code Sprint**: Deep work focus mode.
- **5m Mobility Break**: Guided physical recovery to prevent RSI and burnout.

### 3. Deep Logic Teaching (Gemini Powered) 🧠
- **System-level Persona**: Supportive yet pushing. It won't just give you the answer; it will guide you to the solution.
- **Multimodal Support**: Upload code screenshots or record voice logic explanations for instant analysis.

### 4. Premium Aesthetic 💎
- **Glassmorphism UI**: High-impact depth and blur effects.
- **Deep-Dark Theme**: Engineered for long coding nights.
- **Reactive Animations**: Fluid transitions powered by Framer Motion.

## 🛠️ Tech Stack

- **Stack**: MERN (MongoDB, Express, React, Node.js)
- **AI**: Google Gemini AI (Flash 1.5)
- **Security**: Helmet.js & API Rate Limiting
- **Styling**: Tailwind CSS & Framer Motion

## 🚀 Getting Started

### Prerequisites
- Node.js (v16+)
- MongoDB Instance
- Gemini API Key (AI Studio)

### Installation

1. **Clone & Install**
   ```bash
   npm install # Run from root
   ```

2. **Environment Configuration**
   Create `.env` in the `server` directory:
   ```env
   PORT=5000
   MONGO_URI=your_mongodb_uri
   JWT_SECRET=your_jwt_secret
   GEMINI_API_KEY=your_gemini_api_key
   ```

3. **Launch Execution**
   ```bash
   npm run dev
   ```
